About This Project

A Word Search Game written in PostScript by Rozario Chivers in 1995, for the Computing and Representation class during the first year of study in Software Systems for the Arts and Media.

The game was originally designed for the command-line GhostScript PostScript interpreter on PC. Nearly three decades later, it has been brought to the web using a custom-built PostScript interpreter that renders directly onto HTML Canvas.

How to Play

The game displays a 5 × 5 grid of letters. Hidden inside are 8 networking terms. Find them by clicking the word buttons below the canvas, or by typing the word into the PostScript REPL. Each correct word reveals a red cross-out line on the grid and a letter — find all eight to spell out the hidden message.

Technology

This web application is a from-scratch PostScript interpreter comprising a lexer/parser, stack machine, dictionary system, and a Canvas 2D graphics engine — all written in vanilla JavaScript with no dependencies. The original wordsearch.ps source file runs unmodified.
